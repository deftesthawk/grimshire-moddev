Software and tools

This section covers the software and tools required for Grimshire mod development.

Goal of this step
  • Install the minimum required software for Grimshire modding

Essential

  • Grimshire

    The latest stable game version.

    Wiki
  • BepInEx

    This acts as the foundation layer that loads and manages mods for Unity games. It handles the low-level plumbing required to inject your code into the game at runtime. Installing it also provides Harmony, which enables safe method patching and behavioral changes without modifying the original game files.

    Download
  • Visual Studio

    This is the primary workspace where you’ll write and build your mod. For this guide, you’ll need Visual Studio Community Edition (or an equivalent full IDE). Lightweight editors like Visual Studio Code won’t be sufficient for the workflow used here.

    Download
  • dnSpy

    This tool lets you inspect the game’s compiled code directly. It’s invaluable for understanding how the game works internally and for identifying the classes, methods, and fields your mod will interact with.

    Download

Recommended

  • AssetStudio

    Used to browse and extract assets from Unity games, such as textures, models, audio, and animation data. It’s especially useful when you need to understand how existing assets are structured or when creating new assets that need to match the game’s style and formats.

    Download
  • UnityExplorer

    A BepInEx plugin that gives you real-time access to Unity scenes, objects, components, and values while the game is running. It’s invaluable for experimentation, debugging, and learning how systems behave without constantly rebuilding and relaunching your mod.

    Download
