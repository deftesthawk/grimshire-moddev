Grimshire Mod Development Guide

This guide is still being developed. Additional steps may be added, and existing sections may be revised.

This guide walks you through the complete Grimshire mod workflow: setting up tools, creating a project, configuring builds, adding references, and shipping a working mod.

Who this guide is for
  • Beginners creating their first Grimshire mod
  • Modders who want a concise refresher on tooling, setup, and project structure
  • Anyone who wants a straightforward, step-by-step workflow
What you will build
  • A C# mod project targeting the correct .NET Framework
  • A working plugin that loads through BepInEx
  • A structured workflow for testing, debugging, and release packaging

Guide path

  • Step 1: Software and tools
  • Step 2: Installation and set-up
  • Step 3: Create a Visual Studio project
  • Step 4: Configure assembly and build settings
  • Step 5: Add required references
If you are new to modding, follow the steps in order. Each step builds on the previous one.
