Installation and set-up

This section covers how to install and configure your tools.

Installation and Set-up

When setting up a development environment, it helps to keep everything together in one place. A simple approach is to use a dedicated development folder such as C:\Dev. The exact location doesn’t matter, but it’s important to keep your folder structure consistent and avoid special characters in paths. Stick to letters, numbers, dashes, and underscores to prevent issues.

Example folder structure

Dev/
  Grimshire/
    Tools/
    Mods/
      MyFirstMod/

Visual Studio

Download and run the Visual Studio installer, then select the .NET desktop development workload.

Then open the Individual components tab and select the .NET 4.7.2 targeting pack.

Note that it’s important that you install the .NET 4.7.2 targeting pack. If you don’t, you’ll be unable to select the correct framework when creating your mod project.

dnSpy

Download dnSpy and extract it to a folder location of your choice.

BepInEx Pack

Download the latest version from Nexus and extract it into your Grimshire game folder.
